# Super Trunfo – Países (C++)

Projeto desenvolvido em C++ para simular o jogo Super Trunfo, utilizando comparação de atributos entre cartas.

## Tecnologias utilizadas
- Linguagem C++
- Programação estruturada
- Execução via terminal

## Descrição do projeto
O sistema permite o cadastro de duas cartas contendo informações como estado, cidade, população, área, PIB,
pontos turísticos e densidade demográfica.  
Após o cadastro, o usuário escolhe um atributo para comparação e o programa determina a carta vencedora.

## Conceitos aplicados
- Structs em C++
- Entrada e saída de dados (cin / cout)
- Operadores condicionais (if / else)
- Lógica de comparação
- Cálculo de densidade demográfica

## Funcionalidades
- Cadastro de cartas
- Cálculo automático da densidade
- Comparação dinâmica de atributos
- Exibição do vencedor
